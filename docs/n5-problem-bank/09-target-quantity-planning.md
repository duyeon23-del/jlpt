# N5 탭별 목표 수량 확정 가이드

## 📌 현재 상태
- Phase-1 완료: 125문제 (6개 소분류 × 20문제)
  - 문자·어휘: 한자 읽기(20), 표기(20), 문맥 규정(20)
  - 문법: 판단(20), 배열(20), 문맥(20)
  - 독해: 문맥 5지문 × 4문제(20)

---

## 🎯 선택지: 탭별 목표 수량

### 옵션 A: 가벼운 준비 (총 300문제)
```
문자·어휘 (75문제)
├─ 한자 읽기: 25문제
├─ 표기: 25문제
└─ 문맥 규정: 25문제

문법 (75문제)
├─ 판단: 25문제
├─ 배열: 25문제
└─ 문맥: 25문제

독해 (150문제)
├─ 단문독해: 50문제 (약 10-15 지문)
├─ 중문독해: 50문제 (약 8-10 지문)
└─ 정보검색: 50문제 (약 10-15 지문)
```
